## Project Overview

**Greedy Piggies** is a multiplayer card game built in Unreal Engine 5 and published on Steam, developed as the practical portfolio for this module. My role covered:

- Project coordination and Git branch management
- Python asset scanner tooling for the UE5 pipeline
- Sphinx API documentation deployed via GitHub Pages
- Multiplayer replication systems (auditing mechanic, player visibility, input handling)
- Development commentary and theory essay
